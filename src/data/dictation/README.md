# 听写资料

每份听写是这个文件夹下的**一个 JSON 文件**，应用启动时会自动发现并加入「听写」页的选择列表，**新增听写无需改代码**。

## 命名

文件名即资料标识，建议用听写标题，例如：

```
听写一-3B第十课.json
听写二-3B第十一课.json
```

## 固定格式

```json
{
  "title": "听写二：3B 课本 第十一课",
  "lesson": "3B 课本 第十一课",
  "dateRange": "6月22日 - 6月28日",
  "lang": "zh-CN",
  "words": [
    "一辆车",
    "我们一起去吃饭吧。"
  ]
}
```

字段说明（完整定义见 [`../../features/dictation/types.ts`](../../features/dictation/types.ts)）：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `title` | 是 | 标题，显示在选择列表 |
| `lesson` | 否 | 对应课本/课次 |
| `dateRange` | 否 | 听写日期范围 |
| `lang` | 是 | `"zh-CN"` 或 `"en-US"`，决定朗读语音 |
| `words` | 是 | 听写条目数组，按顺序朗读；每条可为词语、短语或整句 |

> 新增一份听写：复制一个现有 `.json`，改名、改内容即可。
